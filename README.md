## koa2-learning-notes

《我的 Koa2 学习笔记》（koa2-learning-notes）是我学习 Koa2 中记录的学习笔记（持续更新），以 Express 的学习经验，来快速学习 Koa2。
这并不是入门级的书籍，而是需要你具有一定的 Nodejs 基础，ES6 & 7 语法基础，后端开发基础等，才能领会其中的奥义。
